# Commands

Here, you can find a list of all commands that you can use with easyApps. Those can be used on your discord server.

## Applications

Commands that are related to applications.

| Command Name | Description | Permission Level |
| `/create_application` | Starts a process of creating an application. | Application Manager |
| `/edit_application` | Used for editing applications. | Application Manager |
| `/remove_application` | Used for removing applications. | Application Manager |
| `/apply` | Allowes people to apply for an application. | Everyone/People with Selected Role |

## Miscallenous

Commands that can help you with usage of easyApps.

| Command Name | Description | Permission Level |
| `/help` | Shows basic information reagrding easyApps. | Everyone |
| `/server_setup` | Starts a process of setting-up easyApps for a specific server. More can be found in [here][/gettingstarted/setup.md]. | Server Owner |
