# Commands

Here, you can find a list of all commands that you can use with EasyApps. Those can be used on your Discord server.

## Applications

Commands that are related to applications.

||| Command Name
```md
/create_application
```
```md
/edit_application
```
```md
/remove_application
```
```md
/apply
```
||| Description
Starts a process of creating an application.
Used for editing applications.
Used for removing applications.
Allows people to apply for an application.
||| Permission Level
Application Manager
Application Manager
Application Manager
Everyone/People with Selected Role
|||

## Miscallenous

Commands that can help you with the usage of easyApps.

| Command Name | Description | Permission Level |
| ------------------------------ | ------------------------------------------------ |
| `/help` | Shows basic information regarding easyApps. | Everyone |
| `/server_setup` |  | Server Owner |

||| Command Name

```md
/help
```
```md
/server_setup
``` 

||| Description
